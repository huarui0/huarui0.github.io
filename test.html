✨ MySQL Shell
  • ✨ 安装
    • ✨ Installing MySQL Shell with Homebrew 👈 ✨ 采用 本方法 安装 ✨
      • ✨ 安装 步骤
        1. Installing Ruby using Homebrew is easy: 
          ❯ brew search mysql shell
          ❯ brew info mysql-shell
          $ brew install mysql-shell

      • ✨ 安装包 目录的位置
        • 因为 MoOS 不同芯片或（操作）系统，Homebrew 安装目录的位置 不同，安装包的 位置，可以 用 brew info 命令 来 查看： 
          ❯ brew info mysql-shell
    • ✨ 2.3 Installing MySQL Shell on macOS
      • ✨ 安装 步骤
        • Download the package from http://dev.mysql.com/downloads/shell/.
        • Double-click the downloaded DMG to mount it. Finder opens.
        • Double-click the .pkg file shown in the Finder window.
        • Follow the steps in the installation wizard.
        • When the installer finishes, eject the DMG. (It can be deleted.)
